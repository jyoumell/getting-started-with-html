Stage 0: Getting Started with HTML

How the Web Works

Through a computer (and broswer), a user will send a HTTP request to a server. At that point, an HTML document is rendered on the user's machine. The web is made up of billions of these interconnected pages.

HTML is Hypertext Markup Language made up of text content (what you see) and markup (what it looks like). HTML can contain references to other documents (like images and videos) and links to other pages. Take a look at this video for more detail.

Tags & Elements

Elements are larger categorizations that contain HTML tags, which tell the broswer how we would like to render the content within our HTML code.

Types of Elements

HTML elements are either inline or block. Block elements form a box around the content within the tag, which is not visible on a website. Block elements can also contain attributes like width and height.
